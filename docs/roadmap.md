# 后端学习路线

状态说明：`未开始`、`进行中`、`待复习`、`已掌握`。只有学习者能解释并完成相应练习后，才标记为“已掌握”。

## 1. NestJS 与 HTTP 基础（进行中）

- 请求与响应的生命周期
- Module、Controller、Service 与依赖注入
- DTO、参数校验、异常和状态码
- 完成第一个领域化 CRUD

## 2. 数据库基础（未开始）

- PostgreSQL 与关系型数据模型
- ORM、迁移、约束、索引和分页
- 表关系、事务与并发一致性
- Repository 在真实持久化中的作用

## 3. 认证与授权（未开始）

- 密码哈希、Cookie、Session 与 JWT
- Guard、当前用户和 RBAC
- 登录、注册及受保护接口

## 4. Redis 与异步任务（未开始）

- 缓存、过期时间和常见缓存问题
- 验证码、限流、队列和分布式锁

## 5. Docker 与运行环境（未开始）

- Dockerfile、镜像和容器
- 使用 Compose 运行 API、PostgreSQL 和 Redis
- 环境变量与本地开发环境

## 6. 后端工程化与部署（未开始）

- 单元测试、集成测试和端到端测试
- 日志、配置、OpenAPI、监控与错误处理
- CI、生产构建和部署

路线可根据学习反馈调整，不要求机械地一次完成整个阶段。
